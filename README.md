# FDN Mechanistic Research System

This workspace is a deep research project built around **Functional Diagnostic Nutrition (FDN)** — a functional health coaching methodology developed by Reed Davis. The goal is to construct a mechanistic, bidirectional map connecting every FDN lab marker to its upstream causes, downstream symptom outputs, and shared nodes with other markers, then deliver that map as a clinical reasoning framework and an interactive visual tool.

---

## What FDN Is

FDN is a non-medical health coaching system that identifies hidden physiological stressors driving chronic illness. Rather than treating named diseases or suppressing symptoms, it identifies what FDN calls "healing opportunities" — measurable functional imbalances upstream of disease — and addresses them through the **DRESS for Health Success** protocol.

The central concept is **Metabolic Chaos**: the compounding, self-reinforcing dysfunction that occurs when multiple physiological systems are simultaneously stressed. No single system fails in isolation — each dysregulated system loads others, producing cascading, non-linear breakdown that standard medicine misses because it assesses systems separately.

### The Five Core Lab Panels

FDN uses five functional lab tests ordered together as a complete system:

| Panel | Acronym | What It Measures |
|---|---|---|
| Stress & Hormone Profile | SHP | Salivary cortisol (4 time points), DHEA-S, estradiol, progesterone, testosterone, melatonin (noon), secretory IgA |
| Metabolic Wellness Profile | MWP | Indican (maldigestion), urinary bile acids (liver/detox), 8-OHdG (oxidative DNA damage) |
| Mucosal Barrier Assessment | MBA | Zonulin (tight junction integrity), histamine, DAO enzyme |
| GI-MAP Stool Analysis | GI-MAP | Full microbiome and pathogen panel: H. pylori, Candida, parasites, dysbiotic bacteria, commensals, calprotectin, lactoferrin, beta-glucuronidase, anti-gliadin IgA, secretory IgA, occult blood |
| Mediator Release Test | MRT | 170-food immune mediator release panel for food sensitivities |

These five panels are not assessed in isolation. The FDN framework treats them as sequential nodes in a single cascade, each informing the others.

### DRESS for Health Success

Once healing opportunities are identified from the labs, FDN builds a personalized protocol across five lifestyle pillars:

- **D** — Diet (matched to metabolic type via metabolic typing test; guided by MRT food sensitivity results)
- **R** — Rest (circadian rhythm restoration, sleep architecture optimization)
- **E** — Exercise (type and intensity matched to metabolic type and HPA phase)
- **S** — Stress Reduction (identification and elimination of hidden stressors — physical, chemical, emotional, biochemical)
- **S** — Supplementation (targeted to specific lab findings, not generic protocols)

### Hidden Stressors and HEAT

FDN categorizes the upstream causes of metabolic chaos as **HIDDEN stressors** organized under the **HEAT** model:

- **H** — Health (physical stressors: infections, toxins, dysbiosis)
- **E** — Eating (dietary stressors: food sensitivities, nutritional deficiencies, blood sugar dysregulation)
- **A** — Activity (exercise mis-match: too much, too little, or wrong type for the individual's metabolic state)
- **T** — Treating / Stress (psychological and biochemical stress load)

## The Metabolic Chaos Cascade

The cascade architecture is the core organizing principle of FDN. It is bidirectional and self-reinforcing, not linear:

```
Chronic stress
  → HPA axis hyperactivation (elevated cortisol, suppressed DHEA)
  → Catabolic state (cortisol:DHEA ratio elevated beyond 3.8 in females)
  → Suppressed parasympathetic/enteric nervous system
  → Impaired digestive secretions (HCl, enzymes, bile)
  → Maldigestion → dysbiosis (elevated indican)
  → LPS from gram-negative dysbiotic bacteria
  → TLR4 activation → NFκB → pro-inflammatory cytokines
  → Tight junction disruption (elevated zonulin)
  → Intestinal hyperpermeability ("leaky gut")
  → Antigen translocation → systemic immune activation
  → Further HPA re-activation (cytokines stimulate CRH/ACTH)
  → Sustained cortisol → sIgA suppression → pathogen permissiveness
  → Deepened dysbiosis → continued LPS burden
  → Metabolic Chaos (simultaneous multi-system dysfunction)
```

Every FDN lab panel is a measurable node in this loop. The cascade is bidirectional — gut pathology is both a consequence of HPA dysregulation and an independent upstream cause of it.

## Key Biological Mechanisms

These are the specific mechanisms discovered through this research that go beyond standard FDN teaching.

### HPA Axis and the Cortisol:DHEA Ratio

The cortisol:DHEA ratio is FDN's **10th Fundamental Homeostatic Control** (Wolcott). It is the single most powerful integrated measure of catabolic-anabolic balance. FDN female clinical range: 2.2–3.8. Above 3.8 = catabolic dominance, initiating the full metabolic chaos cascade.

Cortisol and DHEA dysregulate through a specific regulatory mechanism — not "pregnenolone steal" (which FDN explicitly debunks). The correct mechanism: ACTH overactivation hyperstimulates the zona fasciculata (cortisol-producing) while cortisol's negative feedback and pro-inflammatory cytokines downregulate 17,20-lyase in the zona reticularis (DHEA-producing). The two zones are anatomically separate and cannot share substrate. The outcome is the same (high cortisol, low DHEA), but the upstream cause (regulatory suppression, not substrate theft) points to different interventions.

HPA dysregulation progresses through stages: cortisol dominance → compensatory state → exhaustion. Each stage produces a distinct SHP pattern and symptom constellation. FDN uses a non-linear dysregulation framework rather than the Selye three-phase model, recognizing that clients can show mixed patterns (some time points high, others low on the same diurnal curve).

### Cortisol's Cross-System Effects

Cortisol is the most cross-systemic variable in the entire FDN map. It simultaneously:
- Downregulates pIgR expression (reducing luminal sIgA delivery)
- Activates IDO (indoleamine 2,3-dioxygenase), shunting tryptophan from serotonin into the kynurenine pathway — generating neuroinflammatory metabolites while depleting serotonin
- Suppresses thyroid T4→T3 conversion (reducing metabolic efficiency)
- Drives zona reticularis suppression (reducing DHEA and sex hormones)
- Suppresses GnRH via CRH, reducing LH/FSH and ovarian sex hormone production
- Suppresses parasympathetic/enteric nervous system function (impairing digestion)
- Increases intestinal permeability directly (reducing tight junction integrity)

### SIgA: A 7-Node Production Axis

Secretory IgA is FDN's primary mucosal immune defense marker. A low sIgA reading is non-specific — the same output can originate from failure at any of seven distinct nodes:

1. **Antigen sampling** — Peyer's patches/M cells fail to sample antigens properly
2. **IgA class switching** — B cells in mesenteric lymph nodes (MLNs) fail to switch to IgA production
3. **Gut homing** — B cells fail to acquire gut-homing receptors (α4β7/CCR9) for trafficking to lamina propria
4. **Plasma cell differentiation** — B cells fail to mature into IgA-secreting plasma cells in the lamina propria
5. **Dimeric IgA production** — plasma cells produce insufficient dimeric IgA + J chain
6. **pIgR transcytosis** — the polymeric immunoglobulin receptor fails to transport dimeric IgA across the epithelium (directly suppressed by cortisol)
7. **SC cleavage and luminal release** — secretory component is not properly cleaved, preventing luminal delivery

Vitamin A (retinoic acid) is rate-limiting at nodes 2, 3, and 4 simultaneously. The BCMO1 enzyme polymorphism reduces beta-carotene-to-retinol conversion by 50–70%, creating hidden Vitamin A insufficiency in plant-dominant diets that standard testing does not catch.

### Noon Melatonin: A Gut Marker, Not a Sleep Marker

FDN measures melatonin at noon on the SHP. Pineal melatonin is near zero at noon. FDN measures noon melatonin to capture gut enterochromaffin (EC) cell melatonin production, which is 400x greater than pineal output. EC cells upregulate melatonin in response to dysbiosis signals (LPS, microbial metabolites). Elevated noon melatonin = gut microbial activation. Low noon melatonin = EC cell exhaustion from chronic gut pathology. The marker is a bridge between SHP adrenal findings and GI-MAP gut pathogen findings.

### Estrobolome and Estrogen Recirculation

A subset of gut bacteria produces beta-glucuronidase, an enzyme that cleaves glucuronide bonds. The liver conjugates estrogens for excretion via bile into stool. Dysbiotic bacteria producing beta-glucuronidase deconjugate these estrogen-glucuronide complexes in the gut lumen, releasing free estrogen that is reabsorbed into circulation. This increases effective estrogen load independent of ovarian production, driving estrogen dominance even when ovarian output is suppressed. GI-MAP beta-glucuronidase elevation is a direct mechanistic explanation for estrogen dominance on the SHP.

### Glyphosate and the Zonulin Pathway

Glyphosate (the active ingredient in Roundup) is 10x more potent than gliadin (gluten protein) at dysregulating the zonulin pathway. Since glyphosate is used as a desiccant on many crops including wheat, gluten and glyphosate frequently co-load in the same foods. Zonulin is the only known physiological regulator of intestinal tight junctions; its dysregulation produces the direct structural permeability that drives the entire downstream cascade.

### Dysbiosis-Permeability Feedback Loop

LPS from gram-negative dysbiotic bacteria binds TLR4 on intestinal epithelial cells → NFκB activation → pro-inflammatory cytokines → downregulation of tight junction proteins (claudin, occludin, ZO-1) → increased paracellular permeability. Simultaneously, those same cytokines suppress GALT sIgA production — the defense mechanism required to clear the dysbiotic organisms generating the damage. This creates a self-sustaining loop: the organisms causing permeability also suppress their own clearance mechanism.

### Histamine-DAO System

DAO (diamine oxidase) is the primary enzyme degrading dietary histamine in the intestinal mucosa. Dysbiosis depletes DAO-producing cells; specific nutrient deficiencies (B6, copper, vitamin C) impair DAO enzyme activity. When DAO is insufficient, dietary histamine accumulates, spills into circulation, and produces systemic effects: vasodilation, headaches, skin flushing, anxiety, gut motility changes. MBA histamine and DAO must be read together — elevated histamine alone is less informative than the histamine:DAO ratio.

### Gut-Brain Chemistry

Clostridia species in the gut produce HPHPA (3-(3-hydroxyphenyl)-3-hydroxypropionic acid), a metabolite measurable on the Organic Acids Test. HPHPA directly competes with dopamine pathways at the enzyme level (dopamine beta-hydroxylase inhibition), producing dopaminergic dysfunction that originates entirely in the gut. This is a direct causal mechanism, not a metaphorical "gut-brain connection."

Cortisol-activated IDO shunts tryptophan into the kynurenine pathway rather than the serotonin/melatonin pathway. This simultaneously depletes serotonin (affecting mood, gut motility, pain sensitivity) and generates neuroinflammatory kynurenine metabolites (quinolinic acid is an NMDA agonist). Both effects are produced by a single enzymatic shift driven by HPA activation.

## Research Architecture

This workspace uses a layered AI-assisted research system called **Cognitive Infrastructure Engineering (CIE)** to conduct and organize the research. The system is governed by a permanent North Star document (`universal-research-system/Research system brain/north-star.md`) that defines absolute operational rules.

### The Eight CIE Frameworks

**HAM — Hierarchical Abstraction Mapping**
Every piece of information is classified at five abstraction levels before any action is taken:
- *meta-meta* — Why the system exists (the worldview behind the work)
- *meta* — How the system is designed to behave (governing principles)
- *macro* — What the system produces (major components and relationships)
- *micro* — How each component works (parameters, variables, behaviors)
- *micro-micro* — Edge cases, implicit constraints, failure modes, unstated assumptions

**ROE — Reverse Outcome Engineering**
Planning always starts from the fully realized end state and decomposes backward until reaching atomic actions that fit in a single context window. Never plan forward from current state.

**PAA — Prompt-as-Architecture**
The structure of a prompt must demonstrate the methodology it wants Claude to follow, not describe it. A prompt structured recursively produces recursive thinking more reliably than one that says "think recursively."

**NSA — North Star Anchoring**
User feedback operates at macro and micro levels. The north star document operates at meta and meta-meta levels. These cannot be conflated. Any user instruction that would alter the meta or meta-meta level of the system triggers a pause and confirmation protocol before any action is taken.

**MSA — Meta-System Architecture**
The system is built in layers: North Star → the product being built → brand/ICP customization → self-iteration command → implementation plan generator → folder/file audit trail. Changes propagate downward only; a lower layer cannot redefine a higher layer.

**CTX — Context-as-Constraint**
The context window is a design parameter, not an inconvenience. Each fresh chat is a stateless execution unit with no memory of previous sessions. All context must be explicitly passed via handoff documents and file references. This forces decomposition, which produces clarity.

**SOC — Separation of Concerns**
Infrastructure (command logic, folder architecture, handoff formats, sequencing) and content (specific research outputs, brand-specific files) are never mixed. Infrastructure changes require explicit authorization. Content can be regenerated freely.

**UAL — Universal Achievement Layer**
An evaluation and feedback layer that governs how systems are measured, diagnosed, and iterated. Adds leverage point discovery to ROE, failure diagnostics to NSA, and phase logic to MSA. Does not change the meta or meta-meta levels of CIE.

### Stateless Execution Units

Every research session is a self-contained execution unit. It receives:
1. Exact file paths to read before doing anything
2. Which CIE frameworks apply
3. A research discernment check (research from source files vs. web vs. internal knowledge)
4. The task, structured per PAA rules
5. An exact output file path and format
6. A handoff document to generate for the next session
7. Instructions telling the user exactly what to do next

Handoff documents contain: current state (with file paths), every decision made and its rationale, rejected options and why, the exact next action, open questions needing user input, and exact paths of every file the next step needs. If any of these six elements is missing, the handoff is incomplete.

### Research Discernment Rule

All research in this system follows a binary decision:
- If the information involves specific software versions, platform behavior, or practices that evolve over time → external research required
- If the information is stable physiology, internal project knowledge, or confirmed cross-session logic → internal context sufficient, no external research

## The Research Question

> For every FDN lab marker in the standard panel, what are the complete bidirectional connections — specifically: (1) every symptom that marker's dysfunction produces and the specific physiological mechanism linking them, (2) every upstream root cause that drives that marker abnormal and its mechanism, and (3) every other FDN marker that shares upstream nodes — such that any input combination of symptoms and lab findings can be traced to the most probable root cause cluster?

Every connection must include a specific physiological mechanism. Vague correlations are excluded.

Reactive Food Burden

**Root Cause / Mechanism Hub Nodes (8)**
HPA Axis Dysregulation Pattern, Pregnenolone Steal and Steroidogenesis Disruption, Gut-Brain-HPA Bidirectional Axis, Dysbiosis-Intestinal Permeability Feedback Loop, Systemic Oxidative Stress Cascade, Hepatic Detoxification Impairment, Histamine-DAO Regulatory System, Mucosal Immune Tolerance and sIgA Function

## The Three End Deliverables

**1. Reference Library**
A complete node entry for every one of the 33 variables. Each entry contains:
- Which symptoms this marker produces when abnormal, with the specific mechanism for each
- Which upstream root causes drive this marker abnormal, with mechanisms
- Which other FDN markers share upstream nodes with this one

**2. Clinical Reasoning Framework**
A practitioner tool where any input combination of symptoms and lab findings can be traced to the most probable root cause cluster. The reasoning process is nodal — patterns across multiple markers localize dysfunction to the most likely failed nodes, then those nodes determine intervention priority.

**3. Interactive Visual Tool**
A single-file interactive HTML application. The practitioner enters multiple symptoms and lab findings simultaneously. The tool identifies the most probable root cause cluster and displays the mechanistic chain connecting inputs to outputs. Every connection in the visual is mechanistically sourced, not correlational.

## The Brain Dump Extraction System

The `bd-extraction/` folder contains the operationalized extraction of a working session that produced partial deliverables (interactive HTML widgets for SIgA and cortisol/DHEA). The extraction system reverse-engineered the session to recover all entities, mechanisms, decisions, and relationships, then organized them into a structured execution plan for completing the remaining work.

The system is organized in four layers:

**00-inventory** — System map of all entities, mechanisms, and relationships in the original conversation. Source of truth before any action is taken.

**01-outcomes** — Registry of all 8 outcomes with dependency maps and fresh-chat vs. chain routing decisions. Includes execution order recommendations.

**02-reverse-engineered** — Step-by-step execution plans for each major deliverable: the HTML visual map, the nodal reasoning framework, the six marker cluster research sessions, and the capstone methodology document.

**03-templates** — Reusable templates for every operation type: marker upstream map files, fresh chat prompts, HTML widget spec (design system + component specs), and cluster research session workflow.

**04-meta-system** — Non-overridable governing rules: approach selection decision tree, parameter extraction algorithm, system integrity rules, and orchestration spec (how all components connect, sequencing rules, handoff contracts between components).

### Integrity Rules

- No marker upstream map is considered complete without a validation checklist pass
- The HTML visual map build is gated on all marker research files being complete — no partial builds
- When a mechanism is uncertain, it is flagged as a research gap and the standard protocol for handling gaps is applied — no guessing
- All FDN marker research is conducted from the source course content first; web research is a fallback only when source content is insufficient
